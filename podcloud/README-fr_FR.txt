== Description ==

Ce plugin permet l'ajout de lecteur intégré podCloud en collant un lien vers un podcast, un épisode, ou une playlist dans l'éditeur WordPress. Il ajoute également le script permettant le redimensionnement magique du lecteur.

== Frequently Asked Questions ==

= Comment ajouter un lecteur podCloud =

Collez un lien podCloud, et WordPress ajoutera automatiquement le lecteur. 
Vous pouvez aussi ajouter un bloc du type podCloud via le bouton + ou la commande /podcloud 

= Quels formats de liens sont acceptés ? =

- `http://podcloud.fr/podcast/*`
- `https://podcloud.fr/podcast/*`
- `http://*.lepodcast.fr/*`
- `https://*.lepodcast.fr/*`
- `http://pdca.st/*`
- `https://pdca.st/*`
- `https://podcloud.fr/users/*/playlists/*`
